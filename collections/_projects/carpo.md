# {{ page.title }}

* 人工智能模型研发平台，用户可通过网页操作便捷完成数据准备、模型训练及预测
* 负责开发 API 后端到 Python 后台服务的异步执行机制，支持模型训练和预测功能
* 在基于 Vue.js 的前端中开发训练配置项设置、训练 log 下载等功能的前端交互

## 功能介绍

模型的创建与管理：提供多种训练任务类型，包括图像分类、物体检测等。自由选择训练数据、预处理操作以及自由设置超参数、损失函数。实时呈现训练状态曲线，可下载完成训练的模型及训练日志。可以对完成训练的模型进行测试。

数据集的上传与管理：将数据集上传至平台并进行管理，可选择与其他用户分享自己的数据集，同时也可以下载其他用户分享的数据集。

组成模块与技术栈：
*	前端：Vue.js 前端框架
*	后端：PHP、Laravel 后端框架 MySQL 数据库
*	训练服务：与具体训练任务有关的工具库或 Python 脚本
